# DormNet-GUI

DormNet-GUI 是一个开源的校园网登录工具，基于 Kotlin Multiplatform 和 Compose Multiplatform 构建，目标是把不同学校的校园网认证流程整理成可复用、可维护的客户端适配。

项目当前仍处于早期阶段，学校适配和平台体验会持续完善。
